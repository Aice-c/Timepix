# 数据分析交接文档：给 5.5 Pro

这份文档用于让研究/论文型模型理解当前 Timepix 课题的数据来源、预处理流程、两个数据集的关系，以及后续论文中应如何设计数据分析章节。

## 1. 研究任务概述

本课题使用 Timepix3 探测器采集到的像素响应矩阵，识别带电粒子的入射极角。输入是单个粒子事件轨迹在探测器像素平面上的二维矩阵，主要模态为：

- `ToT`: Time over Threshold，和局部能量沉积/电荷量相关。
- `ToA`: Time of Arrival，和到达时间/时间结构相关。

当前有两个最终数据集：

- Alpha 数据集：有 `ToT` 和 `ToA`，角度为 `15`, `30`, `45`, `60`。
- C/质子数据集：当前训练主线按 `ToT` 单模态处理，角度更完整，包括 `10`, `20`, `30`, `45`, `50`, `60`, `70`, `80`, `82`, `84`, `86`, `88`, `90`。

## 2. 数据处理三阶段

### 阶段一：原始帧中的轨迹提取

Timepix3 探测器原始数据为 `256 x 256` 矩阵。一个原始帧对应一段时间窗口内的探测器响应，里面可能包含大量粒子事件，不只包含 alpha 或 C/质子，也可能包含其他背景或非目标粒子。

第一步运行轨迹提取程序：

- 对原始 `256 x 256` 矩阵中的激活像素做连通区域分析。
- 将每个连通区域视为一个候选粒子事件轨迹。
- 对连通区域进行几何/质量筛选，去掉明显噪声或无效轨迹。
- 将有效轨迹居中并保存为固定尺寸矩阵，供后续清洗和训练使用。


### 阶段二：基于 ToT 统计量清洗并筛选粒子

轨迹提取后得到的是候选粒子事件，还需要从中筛选出目标粒子数据。用户根据 ToT 数据的事件级统计量清洗数据，核心依据包括：

- 激活像素数 `active_count`
- 激活像素 ToT 和 `active_sum`

这样做的物理依据是：不同粒子或背景事件的能量沉积、轨迹尺寸和激活像素规模可能不同。ToT 与电荷/能量沉积相关，因此适合作为初步筛选依据。

### 阶段三：形成最终监督学习数据集

清洗后得到当前两个监督学习数据集。数据按角度文件夹组织，每个样本是一个粒子事件轨迹矩阵。

Alpha 数据集：

- 当前正式实验主线使用 `Alpha_100`。
- 支持 `ToT` 和 `ToA` 双模态。
- 角度类别：`15`, `30`, `45`, `60`。
- A3 主干模型对比支持 `resnet18_no_maxpool` 作为当前最佳 alpha 主干。
- A4 模态对比显示：当前实现下 `ToT` 单模态优于 raw/log1p `ToT+ToA` early fusion。

C/质子数据集：

- 当前训练主线按 `ToT` 单模态处理。
- 完整角度包括 `10`, `20`, `30`, `45`, `50`, `60`, `70`, `80`, `82`, `84`, `86`, `88`, `90`。
- 用户计划分成两条实验/叙事线：
  - 主模型：使用 `10`, `20`, `30`, `45`, `50`, `60`, `70`，得到一个效果较好的模型。
  - 近垂直角度研究：使用 `80`, `82`, `84`, `86`, `88`, `90`，分析这些角度为什么难以区分。

## 3. 论文中数据集部分建议怎么写

建议把数据集章节写成从物理采集到机器学习样本的完整链条，而不是直接说“我有几个文件夹”。

推荐结构：

1. Timepix3 原始数据说明：
   - 原始帧为 `256 x 256` 像素矩阵。
   - 每个原始帧对应一个时间窗口，可能包含多个粒子事件。
   - ToT/ToA 的物理含义。

2. 单粒子事件提取：
   - 基于激活像素连通区域提取候选轨迹。
   - 每个连通区域对应一个候选粒子事件。
   - 说明为什么需要从多粒子原始帧中切分出单事件。

3. 数据清洗和粒子筛选：
   - 使用 `active_count` 和 `active_sum` 等 ToT 统计量筛选目标粒子。
   - 强调这是物理启发的清洗，不是模型训练后的选择。
   - 如果有阈值或分布图，应放出清洗前后分布。

4. 最终数据集：
   - Alpha 数据集角度、模态、样本数。
   - C/质子数据集角度、模态、样本数。
   - 训练/验证/测试划分策略。
   - 说明 alpha 和 C/质子是两个不同实验数据集，不混合训练。

## 4. 第一部分数据分析：Alpha 与 C/质子常规数据集

目标：证明最终数据集质量可控、类别标签清楚、样本统计合理，并支持后续深度学习实验。

推荐表格：

- 表 1：两个数据集的基本信息。
  - 粒子类型
  - 角度类别
  - 模态
  - 输入矩阵尺寸
  - 每类样本数
  - train/val/test 划分数量

- 表 2：事件级统计特征按角度汇总。
  - `active_count`
  - `active_sum`
  - `active_mean`
  - bbox 宽高/长宽比
  - compactness 或轨迹长度类指标

推荐图：

- 每个角度的代表性 ToT 事件图。
- Alpha 数据集中 ToT/ToA 的代表性样本对比。

- 清洗前后 `active_count` / `active_sum` 分布图。

Alpha 分析重点：

- 说明 `15/30/45/60` 四类角度之间轨迹形态存在可学习差异。
- 因为 alpha 有 ToT/ToA，可以说明多模态实验动机。
- 结合 A4 结果，指出 raw ToA early fusion 未必带来提升，后续 A4b 研究的是 ToA 表达和融合策略。

C/质子常规角度分析重点：

- `10-70` 的轨迹投影长度/形态随角度变化较明显，适合建立分类模型。
- `80/90` 已经出现严重混淆，提示近垂直区域可能达到当前数据表示的角分辨限制。
- 因此把 C/质子研究分为主模型和近垂直极限分析是合理的，而不是“删掉难分类类别”。

## 5. 第二部分数据分析：近垂直 C/质子不可分问题

用户关心的问题：怎样证明深度学习模型无法区分近垂直角度粒子？

建议不要写成绝对证明。更稳妥的论文表述是：

> 在当前探测器设置、事件提取方法、ToT 单模态矩阵表示和已测试模型/特征族条件下，C/质子近垂直角度 `80-90` 度、2 度间隔的数据没有表现出足够的可分性，难以支持可靠监督分类。

### 5.1 已有现象

用户已经观察到：

- C/质子主模型中，`10-70` 表现较好。
- `80` 与 `90` 发生严重混淆。
- 单独取 `80`, `82`, `84`, `86`, `88`, `90` 做近垂直实验时，各种模型架构和超参数都难以学习有效特征。
- 小样本过拟合实验中，每类保留 100 个样本，训练到 50 epoch，模型仍没有过拟合迹象，准确率低于约 17%，接近 6 类随机猜测水平。
- 后续手工特征分析发现邻近角度特征分布高度重叠。

### 5.3 近垂直不可分证据链

建议按“多层证据收敛”的方式写，而不是只放一个模型失败结果。

1. 可视化证据：
   - 展示 `80/82/84/86/88/90` 的代表性 ToT 事件图。
   - 说明近垂直入射时轨迹短、紧凑，视觉上高度相似。

2. 单变量特征重叠：
   - 画 `active_count`, `active_sum`, bbox size, aspect ratio, radial decay 等分布。
   - 使用 box/violin/KDE 图展示邻近角度高度重叠。

3. 统计距离：
   - 使用 KS statistic 或 Wasserstein distance 比较相邻角度特征分布。
   - 注意不要只报告 p-value，因为大样本下极小差异也可能显著。
   - 应强调 effect size 很小，例如 KS 值很低。

4. 多变量特征可分性：
   - PCA/UMAP/t-SNE 可视化。
   - 传统机器学习基线：Random Forest、SVM、Logistic Regression 等。
   - 如果传统特征分类只略高于随机，说明人工设计特征也很难分。

5. 深度学习结果：
   - 多种模型和超参数下准确率仍接近随机。
   - 小样本过拟合实验仍不能过拟合，是很强的信号。
   - 混淆矩阵展示类别之间无稳定边界。

6. 物理解释：
   - 近垂直入射时粒子在像素平面的投影极短。
   - 2 度角差导致的投影长度变化可能小于像素尺度或被电荷扩散/阈值效应淹没。
   - 激活像素主要反映电荷扩散和局部沉积，而不是可分辨的轨迹长度差异。

## 6. 如何阐述 10-70 主模型与 80-90 近垂直研究的决策

不建议写成“因为 80-90 效果差所以删除”。建议写成：

1. C/质子数据覆盖较宽角度范围。
2. 初步实验发现：
   - `10-70` 范围内角度变化带来明显轨迹形态差异，模型能够学习。
   - `80-90` 近垂直范围内轨迹高度紧凑，类别混淆严重。
3. 因此论文将 C/质子任务拆成两个科学问题：
   - 常规角度识别：验证模型在可分角度范围内的性能。
   - 近垂直角分辨极限：研究当前探测器和数据表示是否支持细粒度角度区分。

推荐表述：

> 为避免将可分角度识别问题与近垂直角分辨极限问题混为一谈，本文将 C/质子实验分为两部分。第一部分聚焦于 `10-70` 度范围内的角度分类，评估模型在具有明显轨迹形态差异条件下的识别能力；第二部分单独分析 `80-90` 度近垂直入射区间，探讨当前 ToT 表示下细粒度角度分类的物理和数据限制。
